# Algorithms
A collection of algorithm implementations to practice and learn!

Doing my best to use many languages, not just JS and TS

## Implemented So Far
- [Euclidean](euclidean/) - Greatest common divisor of two integers
- [Elo](elo/) - Tracking rank and expected performance of entities in zero sum games.
- [Sieve of Eratosthenes](sieve-of-eratosthenes/) - Find the prime numbers up to n.
- [Matrix Multiplication](matrix-multiplication) - Multiply Two Matrices.
- [Matrix Determinant](determinant/) - Take the determinant of a matrix.
- [Mergesort](mergesort/) - Sort an array in nlogn.
- [Quicksort](quicksort/) - Another nlogn sorting algorithm.
